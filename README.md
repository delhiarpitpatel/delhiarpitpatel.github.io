# Arpit Patel's Personal Portfolio

This repository contains the source code for my personal portfolio website, built to showcase my skills, projects, and educational journey.

---

### **[View the live site here](https://delhiarpitpatel.github.io/)**

---

### Tech Stack

This portfolio was built from scratch using modern web technologies to be clean, responsive, and performant.

*   **Styling:** [Tailwind CSS](https://tailwindcss.com/) for a utility-first CSS framework.
*   **Animations:** [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/) for subtle scroll-based animations.
*   **Deployment:** Hosted on [GitHub Pages](https://pages.github.com/).

### Running Locally

To view or work on this project locally:

1.  Clone the repository:
    ```bash
    git clone https://github.com/delhiarpitpatel/delhiarpitpatel.github.io.git
    ```
2.  Navigate into the project directory:
    ```bash
    cd delhiarpitpatel.github.io
    ```
3.  Open the `index.html` file in your browser.
